# Python Folder

This folder exists for one reason and one reason only:

**To make using `cat` less painful.**

Instead of opening every file individually like a caveman, you can dump the contents of files directly into your terminal and copy-paste them wherever needed.

## Step 1: Go to the Python folder

```bash
cd python
```

## Step 2: View a single file

```bash
cat filename.py
```

This will print the entire contents of the file to your terminal.

## Step 3: Copy whatever you need

After running the commands above, simply copy the output from your terminal and paste it wherever required.

## Notes

- This folder is **not** meant to be executed directly.
- It exists mainly as a convenient place to store code files.
- Use `cat` to quickly view, reference, or straight-up copy-paste the contents.
- If you somehow manage to get lost in a folder containing only Python files, we may have bigger things to worry about.
